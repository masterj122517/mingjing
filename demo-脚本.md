# 明镜 · Demo 视频录制脚本

> 时长约 6 分钟 | 系统展示 + 旁白

---

## 1. 开场 — 启动 App（0:00-0:30）

```
画面：终端执行 cargo tauri dev → 窗口弹出 → 玻璃拟态暗色主题全貌
     缓慢从上到下扫过：sidebar → 任务列表 → 快速添加栏
```

**旁白：**

> 明镜是一个基于 Tauri 2 + Rust + SQLite 构建的智能学习计划管理器。
> 前端使用纯净的 HTML/CSS/JS，无任何框架依赖，实现了毛玻璃暗色主题的桌面体验。
> 它的核心理念是：设定学习目标 → AI 自动拆解 → 追踪执行进度 → 智能分析反馈。

---

## 2. 设定学习目标（0:30-1:00）

```
画面：点击侧边栏目标区域 → 弹出新建目标弹窗
     输入「通过六级考试」，设描述「目标 550 分」，选截止日期
     点击创建 → 侧边栏出现新目标
```

**旁白：**

> 首先设定一个学习目标。明镜的目标系统支持标题、描述和截止日期。
> 创建后目标会出现在侧边栏，所有关联任务会自动聚合。
> 我们创建一个"通过六级考试"的目标，作为整个学习计划的主线。

---

## 3. AI 智能规划（1:00-1:50）

```
画面：点击侧边栏的目标 → 右上角出现「AI 规划」按钮 → 点击
     输入提示词：「我目前词汇量约 3000，有 30 天时间，每天能学 2 小时」
     点击生成 → 等待 3-5 秒 → AI 返回 8 个结构化任务
     逐一展示：单词积累、听力训练、阅读专项、写作模板、真题模拟……
     用户可以编辑每个任务的标题/优先级/截止日期
```

**旁白：**

> 明镜最具特色的功能——AI 学习计划生成。
> 选择目标后点击 AI 规划，输入你的知识水平和时间安排，
> 大语言模型会根据目标自动拆解为 5-15 个结构化任务，
> 按学习逻辑排序，智能分配优先级和截止日期。
> 你可以在采纳前自由编辑每个任务。

---

## 4. 采纳与调整（1:50-2:30）

```
画面：点击「全部采纳」 → 8 个任务批量创建到任务列表
     点击其中一个任务 → 进入内联编辑 → 改优先级为「高」
     修改截止日期（Todoist 风格迷你日历弹窗）
     添加标签「听力」「真题」
     按分类筛选：创建「英语学习」分类 → 分配任务到该分类
```

**旁白：**

> 采纳后所有任务进入任务列表。每个任务支持内联编辑——
> 点击即可修改标题、优先级、标签、截止日期和分类。
> 分类系统让你可以按科目或类型组织学习内容。
> 标签则允许更细粒度的过滤和标记。

---

## 5. 执行与交互（2:30-3:15）

```
画面：切换视图到「今天」→ 显示今日任务
     勾选一个任务 → 叮咚音效 + 打勾动画 → 任务变灰 + 删除线
     再勾选两个 → 底部出现进度提示
     删除一个任务 → 卡片滑出动画 → Toast「已删除」+ 5 秒撤销倒计时
     点击撤销 → 任务复原
```

**旁白：**

> 明镜提供了丰富的交互反馈。
> 完成任务时有叮咚音效和流畅的完成动画，已完成的文字会以删除线淡化。
> 删除任务时有滑出动画，并提供 5 秒撤销窗口——防止误操作。
> 四种视图切换让你可以从不同角度审视任务：
> 全部任务、今日待办、计划中和已完成。

---

## 6. 统计面板（3:15-4:00）

```
画面：点击侧边栏「📊 统计」按钮 → 切换到统计面板
     展示四个概览数字卡片：总任务 8、已完成 3、待完成 5、已逾期 1
     优先级分布堆叠柱状图（Chart.js）：高/中/低 已完成 vs 待完成
     每日完成趋势折线图：显示近 30 天的完成曲线
     目标进展条：六级目标 3/8 进度 37%
```

**旁白：**

> 统计面板提供了可视化的学习数据。
> 概览卡片一目了然看到整体进度，优先级分布图帮你识别积压情况，
> 每日完成趋势折线图可以回看学习节奏，
> 目标进度条让你清楚每个目标的完成比例。
> 所有图表基于 Chart.js 实现，与毛玻璃主题风格统一。

---

## 7. 导出 PDF 报告（4:00-4:30）

```
画面：点击「导出 PDF 报告」按钮 → 系统保存对话框
     选择保存位置 → 打开生成的 PDF 文件
     PDF 内容：标题「明镜·学习统计报告」+ 日期 + 两张图表截图 + 目标进展文字
```

**旁白：**

> 统计面板的数据可以一键导出为 PDF 报告。
> 报告中包含统计图表截图和目标进展汇总，方便存档或分享给老师。
> 导出基于 jsPDF + html2canvas 实现，在纯前端完成无需服务器。

---

## 8. AI 分析（4:30-5:00）

```
画面：点击返回任务列表 → 点击「AI 分析」
     AI 分析弹窗：风险预警（标红逾期任务「词汇积累已超期 3 天」）
     进度总结「当前共 8 项任务，已完成 3 项，进度 37%」
     建议列表：推荐新任务「增加听力专项练习」
     逐项选择采纳或忽略
```

**旁白：**

> AI 分析功能通过大语言模型对当前进度进行智能诊断。
> 它能识别逾期风险、给出进度总结，并推荐下一步行动。
> 即使没有配置 LLM，明镜也内置了离线规则引擎——
> 自动检测逾期任务和临近截止事项，保证核心功能可用。

---

     GitHub 仓库页面（如果已上传）

## 附录：录制建议

| 项目 | 建议 |
|------|------|
| 录屏工具 | OBS Studio（免费）或系统 QuickTime Player |
| 分辨率 | 1920×1080，窗口居中，800×600 应用内缩放不要超过 100% |
| 准备数据 | 录制前预创建 1-2 个目标、5-8 个任务、2-3 个分类，让画面丰满 |
| 旁白录制 | 用 OBS 同时录屏+麦克风，或后期用剪映/iMovie 配音 |
| BGM | 可选：轻 Lo-fi 背景音乐，音量压低至-20db，不压旁白 |
| 字幕 | 建议加中文字幕（剪映自动生成后修正） |
| 导出格式 | MP4 H.264，1080p 30fps，~50MB |
